# Hotel Table Booking System

## Overview
A Django-based web application that allows customers to reserve tables, browse menus, and manage restaurant bookings.

## Features
- Table reservation system
- Menu browsing
- Customer booking management
- User-friendly interface
- Database-driven application

## Technologies Used
- Python
- Django
- SQLite
- HTML
- CSS
- JavaScript

## Installation

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
